SNCF Time Arrival Project

What to expect

This website is made to make available some information about our project.

You can check the status of our databases. MongoDB, this base is used solely for saving raw data extracted from the api. Dynamo saves all information about scheduled departures and real departures. The relational database monitoring is not implemented but AWS interface is so adapted that it is probably not useful.

The documentation webpage contains some images we showed you during last presentations, (or maybe if you're lucky some information for the next meetings!)

The api part, is an api made to provide schedule and realtime information about trains schedules and delays and provides several endpoints. It will soon provide our own predictions.

The map is a data-visualization of one day of traffic on line H.

The board is a visual interface to query our databases, and ultimately get predictions of delays.
