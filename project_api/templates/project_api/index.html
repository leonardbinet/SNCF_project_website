API parameters

Common parameters to all endpoints

Query parameters:
  • level: int from 0 to at most 4 (depends on the requested object): default 1.
  • query_limit: max number of results of the query: default 10000.
Pagination parameters:
  • limit: max number of results on a page: default 20.
  • offset: number of first result returned: default 0 (starts at beginning).

Specific parameters

Filtering parameters
  • uic_code: filter on specific station: uic code of station (7 or 8 digits)
  • trip_id_filter: filter on specific trip
  • on_route_short_name: filter on specific line
  • on_day: filter on specific day (all endpoints except Routes)
  • active_at_time: filter on trips active at a specific time (hh:mm:ss format, boolean True=> active now)
Realtime information parameters (only for StopTimes endpoint)
  • realtime: boolean: do we update results with realtime information
  • relatime_only: boolean: do we return only results with realtime found
