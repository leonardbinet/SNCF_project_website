Project's API

Overview

You can currently query on given station, on given day (today if not specified) at what time trains really departed, with which delay, and at what time are next trains expected to arrive based on what is displayed on information boards: click here to access to see an example

To access it: api/station/
With parameters:

  • Required: station_id: in a 7 digits format, for instance, station_id=8727622
  • Optional: day: in a yyyymmdd format, for instance, day=20170217. If not specified, today.
